# 30 Days of Shell Scripting

A structured 30-day Bash scripting practice repository focused on Linux automation, scripting fundamentals, and DevOps learning.

This repository contains daily shell scripting exercises and mini-projects created to improve problem-solving and Linux command-line skills.

## How to Run Scripts

Give executable permission:

```bash
chmod +x script.sh
```

Run the script:

```bash
bash script.sh
```

or

```bash
./script.sh
```
